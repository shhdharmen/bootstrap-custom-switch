## 🚀 Getting Started

### ☑️ Minimum Requirements

```sh
node -v
// v10.17.0
git --version
// git version 2.x
```

### ⬇️ Steps to Follow

1. First, fork this repo.
2. Open terminal and:

```sh
git clone <forked-repo-url>
cd bootstrap-theme-kit
npm i
npm run init
npm start
```

3. Browser will open at 3000 port.
4. Start editing your scss/html files and browser will reload.
